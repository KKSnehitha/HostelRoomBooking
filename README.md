This web application allows the students to book hostel rooms through a simple workflow.This system manages creation of rooms,managing student information,room selection,email confirmation and to view the room allocation.

Initially before using the system the hostel rooms are created in the database.

In home page,students can either book a room or else view hostel allocation.

In booking a room,there are 2 steps included:
Step-1:
    Students fill their information along with the selection of type of room.This information is stored temporarily to proceed to next step.
Step-2:
    -The available rooms are displayed from the type of room selected from which a room can be choosed.
    -If there are no rooms available,the syste displays "All rooms of this type are filled".
After successful completion of booking,a succes page is visible along with all the details of student.Additionally,an email is sent to the user for their confirmation.
Here the user can even view their roommates aswell.

In room allocation,The users can view all the rooms along with the room members and the available number of beds vacant and are occupied aswell.

This system provides a complete hostel room booking workflow, including room setup,booking,confirmation and allocation viewing,making hostel management simpler and more efficient.
